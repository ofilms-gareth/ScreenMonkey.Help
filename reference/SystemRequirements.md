# System Requirements
Screen Monkey requires a modern Windows desktop or laptop to run smoothly.

## Hardware
- Intel i5 or equivalent (64 bit)
- GPU score above 800
- Dual monitor outputs (optional, depending on intended use case)

## Software
- Microsoft Windows 10 (64 bit)
- Microsoft .Net Framework 4.5 or newer
- Microsoft PowerPoint 2013 or newer (when using PowerPoint clips)
- Microsoft Edge version 85.0.538.0 or newer (when using web browser, YouTube or Vimeo clips)
